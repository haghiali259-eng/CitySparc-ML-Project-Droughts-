This repository contains the code and results developed throughout the course of my project. To demonstrate the progression of my work, I have included three versions of the model.
1. The original which establishes the baseline neural network structure and training process
2. An updated version that introduces percentage error calculations to improve evaluation and interpretability of model performance
3. The final version, incorporating a stop-loss mechanism to address overfitting and improve generalisation

I have included screenshots of the results produced at different stages of development. These provide a visual comparison of model performance and highlight the impact of each iteration.
These files illustrate the iterative approach taken in this project. They show how the model was progressively refined in response to observed limitations and performance issues.
